## How to Attempt a Challenge

1. Read the challenge document fully before writing any code.
2. Create a branch following the naming convention in the challenge instructions.
3. Write your solution, then write your tests.
4. Run linting and tests locally before submitting.
5. Submit via pull request as described in each challenge.

See [`activities/submissions/README.md`](../submissions/README.md) for submission details.
